# Spring Security

This is a sample project demonstrating the use of Spring Boot and Spring Security to create a secure REST API with JWT authentication.

## Technologies Used

- **Spring Boot**
- **Spring Security**
- **PostgreSQL**
- **JWT (JSON Web Tokens)**
- **H2 Database (for testing)**
- **Maven**
